About this demo

Frontend (HTML) + Java backend serving three endpoints:

  • GET /api/chat?q=... — text answers
  • POST /api/vision (multipart: image, optional q) — image analysis + captioned copy
  • POST /api/gen-image (JSON: {"prompt":"...","size":"1024x1024"}) — image generation

Uploads and generated images live in /public/uploads/.
